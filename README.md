# Hankou 20 Thousand

一个面向现场使用的汉口近代建筑扫街口袋手册。

本项目由个人发烧用户整理，路线骨架参考 B 站 UP 主“艺术与设计史”的“汉口两万步”系列，并补充公开资料、开放状态提示、地点搜索、路线模式和资料来源链接。它不是官方导览，也不代表 UP 主或任何机构立场。

## 使用方式

打开 `index.html`，或通过 GitHub Pages 访问发布页面。

页面支持：

- 地点搜索：按景点、人物、租界、风格、建筑类型等关键词定位。
- 路线模式：完整路线、精华路线、跟着视频走、按建筑类型。
- 现场阅读：每站包含背景、历史故事、现场观察点、开放/预约提示。
- 资料来源：尽量在站点附近附上官方、机构或可查资料链接。

## 资料时效

开放时间、预约方式、运营状态等信息按 2026 年 6 月可查资料整理，后续可能过时。出发前请以官网、官方公众号、预约小程序或现场公告为准。

## 设计原则

- 面向普通游客可用，同时保留理性粉丝复走路线的需求。
- 保持单文件静态网页，不依赖后端、账号、数据库或构建流程。
- 不做打卡、收藏、评论、游戏化和过度粉丝化包装。
- 尊重住户、办公机构、宗教活动和场馆秩序。

## 后续可改进

- 给每站补充更稳定的“30 秒看懂”摘要。
- 将资料来源拆分成单独 `SOURCES.md`，方便维护。
- 增加视频分集时间点索引，但避免大段搬运视频文案。
- 引入轻量地图链接或步行段提示，但不做复杂导航系统。
