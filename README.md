# Hankou 20 Thousand

一个面向现场使用的汉口近代建筑扫街口袋手册。

本项目是非官方整理。路线骨架参考 B 站 UP 主“艺术与设计史”的“汉口两万步”系列，页面补充公开资料、开放状态提示、地点搜索、路线筛选和资料来源链接。它不是官方导览，也不代表原 UP 主或任何机构立场。

在线访问：

- <https://jianghan2thousand-eason-test-d5gzpyo4ce10a4523.webapps.tcloudbase.com>

## 使用方式

打开 `index.html`，或通过上方在线地址访问发布页面。

页面支持：

- 地点搜索：按景点、人物、租界、风格、建筑类型等关键词定位。
- 路线模式：跟视频分集、半日精华、一日精华。
- 现场阅读：每站保留建筑身份、历史背景、可观察要素和开放/预约提示。
- 资料来源：尽量在站点附近附上官方、机构或可查资料链接。
- 高德点位：页面序言处附二维码，可用高德地图导入完整点位。

## 资料时效

开放时间、预约方式、运营状态等信息按 2026 年 6 月可查资料整理，后续可能过时。出发前请以官网、官方公众号、预约小程序或现场公告为准。

## 设计原则

- 面向普通游客可用，同时方便看过视频的观众复走路线。
- 保持单文件静态网页，不依赖后端、账号、数据库或构建流程。
- 不做打卡、收藏、评论、游戏化和过度粉丝化包装。
- 尊重住户、办公机构、宗教活动和场馆秩序。
- 文案尽量客观，不用“教人怎么看”的口吻替代资料本身。
